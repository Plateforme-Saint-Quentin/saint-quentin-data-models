# Smart Irrigation data models

## Description
This repository contains an harmonised NGSI-LD description of the Smart Irrigation data Model. The main NGSI-LD entities of this data are:
-   [City](ngsild-payloads/City.json)

-   [Stadium](/ngsild-payloads/Stadium.json)

-   [Device](/ngsild-payloads/Stadium.json)
	- [Sprinkler](/ngsild-payloads/Stadium.json)
	- [SoilSensor](/ngsild-payloads/Stadium.json)
	- [LawnMower](/ngsild-payloads/Stadium.json)

-   [Valve](/ngsild-payloads/Stadium.json)

-   [Field](/ngsild-payloads/Stadium.json)

-   [SubField](/ngsild-payloads/Stadium.json)
